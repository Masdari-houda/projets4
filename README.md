# MVKL Frontend

Prototype Angular pour une application de diagnostic mammographique.

## Installation

1. Ouvrez un terminal dans `frontend`
2. Exécutez `npm install`
3. Lancez l’application avec `npm start`

## Fonctionnalités prévues

- upload des images MLO et CC
- affichage du BI-RADS estimé
- génération d’un rapport médical
- export PDF / sauvegarde patient

## Remarques

Le front-end contient une simulation de diagnostic. Il faut intégrer un endpoint backend pour connecter le modèle CLIP et retourner les résultats réels.
